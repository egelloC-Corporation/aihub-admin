How do I find my SSH key?
Step 1: Check if you already have one
Open Terminal (Mac) or Command Prompt (Windows) and run:
cat ~/.ssh/id_ed25519.pub or cat ~/.ssh/id_rsa.pub
If you see a key starting with ssh-ed25519 or ssh-rsa, copy the entire line.

Step 2: If you don't have one, create it
Run: ssh-keygen -t ed25519 -C "your-name@egelloc.com"
Press Enter for all prompts. Then copy: cat ~/.ssh/id_ed25519.pub

Important: Only share the .pub file. Never share the file without .pub — that's your private key.
